The range plugin checks if a {\ttfamily Key}\textquotesingle{}s value is within a given range.\hypertarget{md_src_plugins_range_README_autotoc_md432}{}\doxysection{Installation}\label{md_src_plugins_range_README_autotoc_md432}
See \mbox{\hyperlink{doc_INSTALL_md}{installation}}. The package is called {\ttfamily libelektra5-\/experimental}.\hypertarget{md_src_plugins_range_README_autotoc_md433}{}\doxysection{Usage}\label{md_src_plugins_range_README_autotoc_md433}
The plugin checks every {\ttfamily Key} in the {\ttfamily Key\+Set} for the metakey {\ttfamily check/range} which contains either a single range with the syntax {\ttfamily \mbox{[}-\/\mbox{]}min-\/\mbox{[}-\/\mbox{]}max}, or a list of ranges or values separated by {\ttfamily ,} and tests if the {\ttfamily Key}\textquotesingle{}s value is within the range(s).

{\ttfamily check/type} can be used to specify the data type. If not specified otherwise the default value is {\ttfamily long long}

Possible values\+:


\begin{DoxyItemize}
\item {\ttfamily short}, {\ttfamily long}, {\ttfamily long long}

for signed integer values
\item {\ttfamily unsigned short}, {\ttfamily unsigned long}, {\ttfamily unsigned long long}

for unsigned integer values
\item {\ttfamily float}, {\ttfamily double}, {\ttfamily long double}

for floating point values
\item {\ttfamily HEX}

for hexadecimal values
\item {\ttfamily char}

for characters
\end{DoxyItemize}\hypertarget{md_src_plugins_range_README_autotoc_md434}{}\doxysection{Dependencies}\label{md_src_plugins_range_README_autotoc_md434}
