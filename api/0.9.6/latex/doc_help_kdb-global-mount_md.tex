{\ttfamily kdb global-\/mount \mbox{[}$<$plugin$>$ \mbox{[}$<$config$>$\mbox{]} \mbox{[}..\mbox{]}\mbox{]}}


\begin{DoxyItemize}
\item {\ttfamily plugin} are the Elektra plugins to be mounted globally.
\item Plugins may be followed by a {\ttfamily ,} separated list of {\ttfamily keys=values} pairs which will be used as plugin configuration.
\end{DoxyItemize}

{\ttfamily kdb gmount} is an alias and can be used in the same way as {\ttfamily kdb global-\/mount}.\hypertarget{doc_help_kdb-global-mount_md_autotoc_md2393}{}\doxysection{DESCRIPTION}\label{doc_help_kdb-global-mount_md_autotoc_md2393}
This command allows a user to globally mount some plugins that will be part of every interaction with the global keydatabase.\hypertarget{doc_help_kdb-global-mount_md_autotoc_md2394}{}\doxysection{IMPORTANT}\label{doc_help_kdb-global-mount_md_autotoc_md2394}
This command writes into the {\ttfamily /etc} directory and as such it requires root permissions. Use {\ttfamily kdb file system\+:/elektra/globalplugins} to find out where exactly it will write to.\hypertarget{doc_help_kdb-global-mount_md_autotoc_md2395}{}\doxysection{OPTIONS}\label{doc_help_kdb-global-mount_md_autotoc_md2395}
