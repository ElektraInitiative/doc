 \hypertarget{doc_help_kdb-namespace_md_autotoc_md2664}{}\doxysection{SYNOPSIS}\label{doc_help_kdb-namespace_md_autotoc_md2664}
{\ttfamily kdb namespace $<$key name$>$}

Where {\ttfamily key name} is the name of the key.\hypertarget{doc_help_kdb-namespace_md_autotoc_md2665}{}\doxysection{DESCRIPTION}\label{doc_help_kdb-namespace_md_autotoc_md2665}
This command is used to retrieve the namespace of a key, including the eventual trailing \textquotesingle{}\+:\textquotesingle{}. For the cascading namespace, the empty string is returned.\hypertarget{doc_help_kdb-namespace_md_autotoc_md2666}{}\doxysection{RETURN VALUES}\label{doc_help_kdb-namespace_md_autotoc_md2666}
This command will return the following values as an exit status\+:


\begin{DoxyItemize}
\item 0\+: No errors.
\item 7\+: invalid key name passed, see \mbox{\hyperlink{doc_help_kdb_md}{kdb(1)}} for a list of return codes used by kdb.
\end{DoxyItemize}\hypertarget{doc_help_kdb-namespace_md_autotoc_md2667}{}\doxysection{OPTIONS}\label{doc_help_kdb-namespace_md_autotoc_md2667}

\begin{DoxyItemize}
\item {\ttfamily -\/n}, {\ttfamily -\/-\/no-\/newline}\+: Suppress the newline at the end of the output.
