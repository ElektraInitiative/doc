{\ttfamily kdb restore $<$timestamp$>$}\hypertarget{doc_help_kdb-restore_md_autotoc_md2518}{}\doxysection{DESCRIPTION}\label{doc_help_kdb-restore_md_autotoc_md2518}
This command will restore everything backed up by {\ttfamily kdb backup} (or stashed by {\ttfamily kdb stash}) where {\ttfamily timestamp} is the timestamp returned by {\ttfamily kdb backup} (or {\ttfamily kdb stash}).\hypertarget{doc_help_kdb-restore_md_autotoc_md2519}{}\doxysection{WARNING}\label{doc_help_kdb-restore_md_autotoc_md2519}
These changes cannot be undone, please use with care! This command writes into the {\ttfamily /etc} directory and as such it requires root permissions.\hypertarget{doc_help_kdb-restore_md_autotoc_md2520}{}\doxysection{EXAMPLES}\label{doc_help_kdb-restore_md_autotoc_md2520}
