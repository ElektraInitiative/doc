 Usually, a backend refers to exactly one file per namespace. This file can be returned using {\ttfamily kdb file}.\hypertarget{doc_decisions_multiple_file_backends_md_autotoc_md2018}{}\doxysection{Problem}\label{doc_decisions_multiple_file_backends_md_autotoc_md2018}
In some situations a single mountpoint refers to more than one file per namespace\+:


\begin{DoxyItemize}
\item For XDG in the {\ttfamily system} namespace may contain several files (XDG\+\_\+\+CONFIG\+\_\+\+DIRS).
\item A fallback file if some data cannot be stored in some format (Idea from @kodebach\+: writing the same content to several files, merging when reading)
\end{DoxyItemize}\hypertarget{doc_decisions_multiple_file_backends_md_autotoc_md2019}{}\doxysection{Constraints}\label{doc_decisions_multiple_file_backends_md_autotoc_md2019}
\hypertarget{doc_decisions_multiple_file_backends_md_autotoc_md2020}{}\doxysection{Assumptions}\label{doc_decisions_multiple_file_backends_md_autotoc_md2020}
\hypertarget{doc_decisions_multiple_file_backends_md_autotoc_md2021}{}\doxysection{Considered Alternatives}\label{doc_decisions_multiple_file_backends_md_autotoc_md2021}
\hypertarget{doc_decisions_multiple_file_backends_md_autotoc_md2022}{}\doxysection{Decision}\label{doc_decisions_multiple_file_backends_md_autotoc_md2022}
Multiple File Backends are not supported for now in the case of writing files.

Multiple sources in one namespace only work, if the fallback Key\+Set is part of the mountpoint config. That way any change to the fallback Key\+Set would essentially make the whole thing a different mountpoint and thereby invalidate all guarantees.\hypertarget{doc_decisions_multiple_file_backends_md_autotoc_md2023}{}\doxysection{Rationale}\label{doc_decisions_multiple_file_backends_md_autotoc_md2023}
Writeable multiple file backends would\+:


\begin{DoxyItemize}
\item make it impossible for admins to modify content of all files using Elektra
\item do not work atomically (a journal would be needed)
\item do not work together with mmap (as it only checks one file for cache misses)
\end{DoxyItemize}\hypertarget{doc_decisions_multiple_file_backends_md_autotoc_md2024}{}\doxysection{Implications}\label{doc_decisions_multiple_file_backends_md_autotoc_md2024}
