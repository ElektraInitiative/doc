The Lineendings Plugin verifies the line endings of a file. If inconsistent line endings or line endings that don\textquotesingle{}t match {\ttfamily valid} are detected the plugin yields an error.\hypertarget{md_src_plugins_lineendings_README_autotoc_md294}{}\doxysection{Installation}\label{md_src_plugins_lineendings_README_autotoc_md294}
See \mbox{\hyperlink{doc_INSTALL_md}{installation}}. The package is called {\ttfamily libelektra5-\/extra}.\hypertarget{md_src_plugins_lineendings_README_autotoc_md295}{}\doxysection{Configuration}\label{md_src_plugins_lineendings_README_autotoc_md295}
{\ttfamily valid} The key tells the plugin to reject all line endings other than specified in this key. Valid options\+: CRLF, LFCR, CR, LF If the key doesn\textquotesingle{}t exist only inconsistent line endings are rejected.
