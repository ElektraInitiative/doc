This plugin validates IP addresses using regular expressions.\hypertarget{md_src_plugins_ipaddr_README_autotoc_md259}{}\doxysubsection{Installation}\label{md_src_plugins_ipaddr_README_autotoc_md259}
See \mbox{\hyperlink{doc_INSTALL_md}{installation}}. The package is called {\ttfamily libelektra5-\/experimental}.\hypertarget{md_src_plugins_ipaddr_README_autotoc_md260}{}\doxysubsection{Usage}\label{md_src_plugins_ipaddr_README_autotoc_md260}

\begin{DoxyCode}{0}
\DoxyCodeLine{\# Mount `ipaddr` plugin to cascading namespace `/tests/ipaddr`}
\DoxyCodeLine{kdb mount config.dump /tests/ipaddr dump ipaddr}
\DoxyCodeLine{}
\DoxyCodeLine{\# Check the validity of the IP stored in `/tests/ipaddr/ipv4`}
\DoxyCodeLine{kdb meta-\/set /tests/ipaddr/ipv4 check/ipaddr ipv4}
\DoxyCodeLine{}
\DoxyCodeLine{\# Try to set an incorrect IP address}
\DoxyCodeLine{kdb set /tests/ipaddr/ipv4 127.0.0.1337}
\DoxyCodeLine{\# STDERR: .*Validation Semantic.*}
\DoxyCodeLine{\# ERROR:  C03200}
\DoxyCodeLine{\# RET:    5}
\DoxyCodeLine{}
\DoxyCodeLine{\# Set a correct IPv4 address}
\DoxyCodeLine{kdb set /tests/ipaddr/ipv4 127.0.0.1}
\DoxyCodeLine{kdb get /tests/ipaddr/ipv4}
\DoxyCodeLine{\#> 127.0.0.1}
\DoxyCodeLine{}
\DoxyCodeLine{\# By default the plugin allows both IPv4 and IPv6 addresses}
\DoxyCodeLine{kdb meta-\/set /tests/ipaddr/address check/ipaddr "{}"{}}
\DoxyCodeLine{}
\DoxyCodeLine{\# Set correct IP addresses}
\DoxyCodeLine{kdb set /tests/ipaddr/address 1.2.3.4}
\DoxyCodeLine{kdb set /tests/ipaddr/address ::1}
\DoxyCodeLine{}
\DoxyCodeLine{\# Try to set incorrect addresses}
\DoxyCodeLine{kdb set /tests/ipaddr/address bad::ip}
\DoxyCodeLine{\# RET: 5}
\DoxyCodeLine{kdb set /tests/ipaddr/address 1.2.-\/3.4}
\DoxyCodeLine{\# RET: 5}
\DoxyCodeLine{}
\DoxyCodeLine{\# Undo modifications to the database}
\DoxyCodeLine{kdb rm -\/r /tests/ipaddr}
\DoxyCodeLine{kdb umount /tests/ipaddr}

\end{DoxyCode}
\hypertarget{md_src_plugins_ipaddr_README_autotoc_md261}{}\doxysubsection{Limitations}\label{md_src_plugins_ipaddr_README_autotoc_md261}
The plugin only checks IP addresses for validity. It is not able to resolve hostnames. If you are looking for a plugin that supports hostnames, check out the \mbox{\hyperlink{md_src_plugins_network_README_src_plugins_network_README_md}{network plugin}}.
