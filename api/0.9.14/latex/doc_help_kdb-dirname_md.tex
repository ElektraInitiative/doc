{\ttfamily kdb dirname $<$key name$>$}

Where {\ttfamily key name} is the name of the key.\hypertarget{doc_help_kdb-dirname_md_autotoc_md2847}{}\doxysection{DESCRIPTION}\label{doc_help_kdb-dirname_md_autotoc_md2847}
This command is used to retrieve the directory name of a key, without any namespace.\hypertarget{doc_help_kdb-dirname_md_autotoc_md2848}{}\doxysection{RETURN VALUES}\label{doc_help_kdb-dirname_md_autotoc_md2848}
This command will return the following values as an exit status\+:


\begin{DoxyItemize}
\item 0\+: No errors.
\item 7\+: invalid key name passed, see \mbox{\hyperlink{doc_help_kdb_md}{kdb(1)}} for a list of return codes used by kdb.
\end{DoxyItemize}\hypertarget{doc_help_kdb-dirname_md_autotoc_md2849}{}\doxysection{OPTIONS}\label{doc_help_kdb-dirname_md_autotoc_md2849}

\begin{DoxyItemize}
\item {\ttfamily -\/n}, {\ttfamily -\/-\/no-\/newline}\+: Suppress the newline at the end of the output.
