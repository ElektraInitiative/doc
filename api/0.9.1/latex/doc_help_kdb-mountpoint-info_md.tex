{\ttfamily kdb mountpoint-\/info \mbox{[}mount point\mbox{]}}

This command will print information about the version, default resolver and default storage plugin. When a mount point is given as argument additional informations about the mount point (e.\+g. configuration) is printed.


\begin{DoxyCode}{0}
\DoxyCodeLine{kdb mountpoint-info}
\DoxyCodeLine{\#> Version: 0.8.19}
\DoxyCodeLine{\#> Default resolver: resolver\_fm\_hpu\_b}
\DoxyCodeLine{\#> Default storage: ini}
\DoxyCodeLine{}
\DoxyCodeLine{kdb mount /tmp/test.ini system/test ini hello=ini -c hello=world}
\DoxyCodeLine{}
\DoxyCodeLine{kdb mountpoint-info system/test}

\DoxyCodeLine{\#> Default resolver: resolver\_fm\_hpu\_b}
