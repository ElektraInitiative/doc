You can use {\ttfamily scripts/copy-\/template} to automatically rename everything to your plugin name\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{cd src/plugins}
\DoxyCodeLine{../../scripts/copy-\/template yourplugin}

\end{DoxyCode}


Then update the README.\+md of your newly created plugin\+:


\begin{DoxyItemize}
\item enter your full name+email in {\ttfamily infos/author}
\item make sure {\ttfamily status}, {\ttfamily placements}, and other clauses conform to descriptions in {\ttfamily doc/\+CONTRACT.\+ini}
\item update the one-\/line description above
\item add your plugin in {\ttfamily src/plugins/\+README.\+md}
\item and rewrite the rest of this {\ttfamily README.\+md} to give a great explanation of what your plugin does
