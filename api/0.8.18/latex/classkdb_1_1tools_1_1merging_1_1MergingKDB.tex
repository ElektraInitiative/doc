\subsection{Detailed Description}
Provides a merging wrapper around a \hyperlink{classkdb_1_1KDB}{K\+D\+B} instance. 

The wrapper allows to pass a three way merger instance that is used to resolve conflicts during \hyperlink{classkdb_1_1KDB}{K\+D\+B} set.
