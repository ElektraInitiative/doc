This plugin is a storage plugin that will use the syscall {\ttfamily uname (2)}. No resolver is needed for that plugin to work.\hypertarget{md_src_plugins_uname_README_autotoc_md575}{}\doxysection{Installation}\label{md_src_plugins_uname_README_autotoc_md575}
See \mbox{\hyperlink{doc_INSTALL_md}{installation}}. The package is called {\ttfamily libelektra5-\/extra}.\hypertarget{md_src_plugins_uname_README_autotoc_md576}{}\doxysection{Special Values}\label{md_src_plugins_uname_README_autotoc_md576}
This plugin defines following keynames below its mount point\+:


\begin{DoxyItemize}
\item sysname
\item nodename
\item release
\item version
\item machine
\end{DoxyItemize}\hypertarget{md_src_plugins_uname_README_autotoc_md577}{}\doxysection{Restrictions}\label{md_src_plugins_uname_README_autotoc_md577}
This plugin is read-\/only.\hypertarget{md_src_plugins_uname_README_autotoc_md578}{}\doxysection{Example}\label{md_src_plugins_uname_README_autotoc_md578}
