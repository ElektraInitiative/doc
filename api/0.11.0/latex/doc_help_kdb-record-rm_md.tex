{\ttfamily kdb record-\/rm $<$key$>$}~\newline
\hypertarget{doc_help_kdb-record-rm_md_autotoc_md3076}{}\doxysection{DESCRIPTION}\label{doc_help_kdb-record-rm_md_autotoc_md3076}
This command removes the specified {\ttfamily key} from the recording session. As far as the recording session is concerned, it\textquotesingle{}s as if the key was never changed. If the key is changed again after it was removed from the session, it will be put back into the session.\hypertarget{doc_help_kdb-record-rm_md_autotoc_md3077}{}\doxysection{OPTIONS}\label{doc_help_kdb-record-rm_md_autotoc_md3077}

\begin{DoxyItemize}
\item {\ttfamily -\/H}, {\ttfamily -\/-\/help}\+: Show the man page.
\item {\ttfamily -\/V}, {\ttfamily -\/-\/version}\+: Print version info.
\item {\ttfamily -\/p}, {\ttfamily -\/-\/profile $<$profile$>$}\+: Use a different kdb profile.
\item {\ttfamily -\/C}, {\ttfamily -\/-\/color $<$when$>$}\+: Print never/auto(default)/always colored output.
\item {\ttfamily -\/r}, {\ttfamily -\/-\/recursive}\+: Work in a recursive mode. Will also remove all keys below {\ttfamily key}.
\item {\ttfamily -\/v}, {\ttfamily -\/-\/verbose}\+: Explain what is happening. Prints additional information in case of errors/warnings.
\item {\ttfamily -\/d}, {\ttfamily -\/-\/debug}\+: Give debug information. Prints additional debug information in case of errors/warnings.
\end{DoxyItemize}\hypertarget{doc_help_kdb-record-rm_md_autotoc_md3078}{}\doxysection{RETURN VALUE}\label{doc_help_kdb-record-rm_md_autotoc_md3078}

\item 0\+: Successful.
\item 1-\/10\+: standard exit codes, see \mbox{\hyperlink{doc_help_kdb_md}{kdb(1)}}.s
\item 11\+: An error occurred during removal of the key(s).
