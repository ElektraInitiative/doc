\subsection*{D\+E\+S\+C\+R\+I\+P\+T\+I\+ON}

This command prints which file a given key is stored in. While many keys are stored in a default key database file, many others are stored in any number of configuration files located all over the system. This tool is made to allow users to find out the file that a key is actually stored in. This command makes use of Elektraâ€™s {\ttfamily resolver} plugin which the uer can learn more about by running the command {\ttfamily kdb info resolver}.
