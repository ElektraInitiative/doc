 \hypertarget{doc_decisions_6_implemented_relative_md_autotoc_md2450}{}\doxysection{Problem}\label{doc_decisions_6_implemented_relative_md_autotoc_md2450}
There is a different behavior of various plugins whether their name is absolute or relative, including\+:


\begin{DoxyEnumerate}
\item mounting the same file somewhere else does not work
\item importing somewhere else (other than from where it was exported) does not work (See \href{https://github.com/ElektraInitiative/libelektra/issues/51}{\texttt{ here}})
\end{DoxyEnumerate}\hypertarget{doc_decisions_6_implemented_relative_md_autotoc_md2451}{}\doxysection{Constraints}\label{doc_decisions_6_implemented_relative_md_autotoc_md2451}

\begin{DoxyItemize}
\item at least the dump plugin must be able to handle its old files
\end{DoxyItemize}\hypertarget{doc_decisions_6_implemented_relative_md_autotoc_md2452}{}\doxysection{Assumptions}\label{doc_decisions_6_implemented_relative_md_autotoc_md2452}

\begin{DoxyItemize}
\item it still will be easy to support a workflow that exports/imports everything
\item mounting across namespaces (user/system) does not make sense
\end{DoxyItemize}\hypertarget{doc_decisions_6_implemented_relative_md_autotoc_md2453}{}\doxysection{Considered Alternatives}\label{doc_decisions_6_implemented_relative_md_autotoc_md2453}

\begin{DoxyItemize}
\item allow relative/absolute plugins and mark them what they are, tools (e.\+g. import/export) use this knowledge and react accordingly. This would still not solve problem 1.)
\end{DoxyItemize}\hypertarget{doc_decisions_6_implemented_relative_md_autotoc_md2454}{}\doxysection{Decision}\label{doc_decisions_6_implemented_relative_md_autotoc_md2454}
Key names shall be relative to parent key name.\hypertarget{doc_decisions_6_implemented_relative_md_autotoc_md2455}{}\doxysection{Rationale}\label{doc_decisions_6_implemented_relative_md_autotoc_md2455}
Provides a better import/export/remount and also a more uniform experience between different plugins.\hypertarget{doc_decisions_6_implemented_relative_md_autotoc_md2456}{}\doxysection{Implications}\label{doc_decisions_6_implemented_relative_md_autotoc_md2456}
