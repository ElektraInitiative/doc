\subsection{Detailed Description}
The private trie structure. 

A trie is a data structure which can handle the longest prefix matching very fast. This is exactly what needs to be done when using \hyperlink{group__kdb_ga28e385fd9cb7ccfe0b2f1ed2f62453a1}{kdb\+Get()} and \hyperlink{group__kdb_ga11436b058408f83d303ca5e996832bcf}{kdb\+Set()} in a hierarchy where backends are mounted -\/ you need the backend mounted closest to the parent\+Key.
