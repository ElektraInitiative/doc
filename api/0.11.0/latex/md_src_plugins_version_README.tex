This plugin is very simple. It implements the minimum functionality required to qualify as a read-\/only backend plugin.

The data this plugin returns during {\ttfamily \mbox{\hyperlink{group__kdb_ga28e385fd9cb7ccfe0b2f1ed2f62453a1}{kdb\+Get()}}} is simple the data returned by {\ttfamily elektra\+Version\+Key\+Set()} with each {\ttfamily Key} additionally marked with the metadata {\ttfamily restrict/write} and {\ttfamily restrict/remove}.
