Assigns the vertices of the r-\/uniform r-\/partite hypergraph. 

Allocs the memory for the final OPMPHM {\ttfamily Opmphm-\/$>$graph}. Uses the remove sequence {\ttfamily Opmphm\+Graph-\/$>$remove\+Sequence}, generated during cycle check, to assign each vertex. Either with {\ttfamily Opmphm\+Edge-\/$>$order} or the default order, default is the order of {\ttfamily Opmphm\+Init-\/$>$data}.
