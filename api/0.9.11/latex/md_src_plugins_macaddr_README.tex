This plugin validates MAC-\/addresses. The following MAC-\/address-\/formats are supported\+: \begin{DoxyVerb}XX-XX-XX-XX-XX-XX
XX:XX:XX:XX:XX:XX
XXXXXX-XXXXXX
Integer values (0 - 281474976710655)
\end{DoxyVerb}
 {\ttfamily kdb\+Get} returns an integer-\/representation of these values.\hypertarget{md_src_plugins_macaddr_README_autotoc_md333}{}\doxysection{Installation}\label{md_src_plugins_macaddr_README_autotoc_md333}
