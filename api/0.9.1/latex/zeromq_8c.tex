Attach to Zero\+Mq socket. 

The callback is called whenever socket is readable or writable (depending on type) and data can be processed. The callback should not do blocking calls (e.\+g. use Z\+M\+Q\+\_\+\+D\+O\+N\+T\+W\+A\+IT for zmq\+\_\+$\ast$recv()). Since Zero\+Mq guarantees that multipart messages arrive at once, data will be available. New adapter instances are enabled.
