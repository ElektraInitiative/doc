The range plugin checks if a {\ttfamily Key}\textquotesingle{}s value is within a given range.\hypertarget{md_src_plugins_range_README_autotoc_md446}{}\doxysection{Installation}\label{md_src_plugins_range_README_autotoc_md446}
See \mbox{\hyperlink{doc_INSTALL_md}{installation}}. The package is called {\ttfamily libelektra5-\/experimental}.\hypertarget{md_src_plugins_range_README_autotoc_md447}{}\doxysection{Usage}\label{md_src_plugins_range_README_autotoc_md447}

\begin{DoxyItemize}
\item The plugin checks every {\ttfamily Key} in the {\ttfamily Key\+Set} having a metakey {\ttfamily check/range}.
\item For these keys, it checks whether the key value is within the specified range.
\item {\ttfamily check/range} can contain either\+:
\begin{DoxyEnumerate}
\item a single range with the syntax {\ttfamily \mbox{[}-\/\mbox{]}min-\/\mbox{[}-\/\mbox{]}max}
\item or a list of ranges or values separated by {\ttfamily ,}
\end{DoxyEnumerate}
\item Metakey {\ttfamily check/type} can be used to specify the data type. If not specified, metakey {\ttfamily type} will be used as fallback. If both are unspecified or unsupported, the type is assumed to be {\ttfamily long long}.
\end{DoxyItemize}

Possible values\+:


\begin{DoxyItemize}
\item {\ttfamily short}, {\ttfamily long}, {\ttfamily long long}

for signed integer values
\item {\ttfamily unsigned short}, {\ttfamily unsigned long}, {\ttfamily unsigned long long}

for unsigned integer values
\item {\ttfamily float}, {\ttfamily double}, {\ttfamily long double}

for floating point values
\item {\ttfamily HEX}

for hexadecimal values
\item {\ttfamily char}

for characters
\end{DoxyItemize}\hypertarget{md_src_plugins_range_README_autotoc_md448}{}\doxysection{Dependencies}\label{md_src_plugins_range_README_autotoc_md448}
