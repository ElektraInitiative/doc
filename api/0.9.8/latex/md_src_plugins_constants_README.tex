Includes constants information into the key database. The constants are defined during CMake build.

The plugin is readonly.\hypertarget{md_src_plugins_constants_README_autotoc_md96}{}\doxysection{Usage}\label{md_src_plugins_constants_README_autotoc_md96}
To mount it, use


\begin{DoxyCode}{0}
\DoxyCodeLine{kdb mount -\/R noresolver none user:/tests/constants constants}

\end{DoxyCode}
