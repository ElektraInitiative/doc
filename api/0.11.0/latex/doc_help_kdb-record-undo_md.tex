{\ttfamily kdb record-\/undo \mbox{[}$<$parent\+\_\+key$>$\mbox{]}}~\newline
\hypertarget{doc_help_kdb-record-undo_md_autotoc_md3097}{}\doxysection{DESCRIPTION}\label{doc_help_kdb-record-undo_md_autotoc_md3097}
This command will undo all the changes that have been recorded. The optional parameter {\ttfamily parent\+\_\+key} can be used to restrict the undo operation to a specific subtree of the KDB. After execution, the recording session will no longer contain the undone keys.\hypertarget{doc_help_kdb-record-undo_md_autotoc_md3098}{}\doxysection{OPTIONS}\label{doc_help_kdb-record-undo_md_autotoc_md3098}
