{\ttfamily kdb set $<$key name$>$ \mbox{[}$<$value$>$\mbox{]}}

Where {\ttfamily key name} is the name of the key you wish to set the value of (or create) and {\ttfamily value} is the value you would like to set the key to. If the {\ttfamily value} argument is not passed, the key will be set to a value of {\ttfamily null}.\hypertarget{doc_help_kdb-set_md_autotoc_md1834}{}\doxysection{D\+E\+S\+C\+R\+I\+P\+T\+I\+ON}\label{doc_help_kdb-set_md_autotoc_md1834}
This command allows the user to set the value of an individual key.\hypertarget{doc_help_kdb-set_md_autotoc_md1835}{}\doxysection{E\+M\+P\+T\+Y V\+A\+L\+U\+ES}\label{doc_help_kdb-set_md_autotoc_md1835}
To set a key to an empty value, {\ttfamily \char`\"{}\char`\"{}} should be passed for the {\ttfamily value} argument.\hypertarget{doc_help_kdb-set_md_autotoc_md1836}{}\doxysection{N\+E\+G\+A\+T\+I\+V\+E V\+A\+L\+U\+ES}\label{doc_help_kdb-set_md_autotoc_md1836}
