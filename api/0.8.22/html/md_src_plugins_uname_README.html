This plugin is a storage plugin that will use the syscall uname (2). No resolver is needed for that plugin to work.

Special Values

This plugin defines following keynames below its mountpoint:

  • sysname
  • nodename
  • release
  • version
  • machine

Restrictions

This plugin is read-only.

Example

To mount uname information using this plugin: kdb mount -R noresolver none system/uname uname
