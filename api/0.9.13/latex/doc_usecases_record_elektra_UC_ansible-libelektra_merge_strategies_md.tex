\item {\bfseries{Title\+:}} Select merge-\/strategy in ansible-\/libelektra
\item {\bfseries{Scope\+:}} ansible-\/libelektra
\item {\bfseries{Level\+:}} User goal
\item {\bfseries{Actors\+:}} User (usually sysadmin)
\item {\bfseries{Brief\+:}} It should be possible to specify the merge strategy used for conflicts. The merging should be performed by ansible-\/libelektra. The following strategies should be implemented\+:
\begin{DoxyItemize}
\item OURS\+: use the new values
\item THEIRS\+: keep the old values
\item ABORT\+: on conflict don\textquotesingle{}t merge and report an error
\end{DoxyItemize}
