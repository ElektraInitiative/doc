{\ttfamily kdb record-\/state}~\newline
\hypertarget{doc_help_kdb-record-state_md_autotoc_md3086}{}\doxysection{DESCRIPTION}\label{doc_help_kdb-record-state_md_autotoc_md3086}
This command prints out information about the current state of the recording session. This includes\+:


\begin{DoxyItemize}
\item Whether recording is currently enabled or not
\item What parts of the KDB will be recorded in the future. (The session may already contain changes from other parts of the KDB)
\item How many keys have been recorded
\item Which keys have been recorded
\end{DoxyItemize}

For now, the output format is not considered stable and may change over time.\hypertarget{doc_help_kdb-record-state_md_autotoc_md3087}{}\doxysection{OPTIONS}\label{doc_help_kdb-record-state_md_autotoc_md3087}

\begin{DoxyItemize}
\item {\ttfamily -\/H}, {\ttfamily -\/-\/help}\+: Show the man page.
\item {\ttfamily -\/V}, {\ttfamily -\/-\/version}\+: Print version info.
\item {\ttfamily -\/p}, {\ttfamily -\/-\/profile $<$profile$>$}\+: Use a different kdb profile.
\item {\ttfamily -\/C}, {\ttfamily -\/-\/color $<$when$>$}\+: Print never/auto(default)/always colored output.
\item {\ttfamily -\/v}, {\ttfamily -\/-\/verbose}\+: Explain what is happening. Prints additional information in case of errors/warnings.
\item {\ttfamily -\/d}, {\ttfamily -\/-\/debug}\+: Give debug information. Prints additional debug information in case of errors/warnings.
\end{DoxyItemize}\hypertarget{doc_help_kdb-record-state_md_autotoc_md3088}{}\doxysection{RETURN VALUE}\label{doc_help_kdb-record-state_md_autotoc_md3088}

\begin{DoxyItemize}
\item 0\+: Successful.
\item 1-\/10\+: standard exit codes, see \mbox{\hyperlink{doc_help_kdb_md}{kdb(1)}}
\item 11\+: An error occurred during gathering the state information.
