 \hypertarget{doc_decisions_commit_function_md_autotoc_md1707}{}\doxysection{Problem}\label{doc_decisions_commit_function_md_autotoc_md1707}
When {\ttfamily \mbox{\hyperlink{group__kdb_ga11436b058408f83d303ca5e996832bcf}{kdb\+Set()}}} is called, plugins implementing the commit role need to track their state to distinguish between carrying out that role and carrying out potential other roles (commit and setresolver for the resolver plugin, for example). This limits the possibilities of plugin reuse and the ways plugins can be combined.\hypertarget{doc_decisions_commit_function_md_autotoc_md1708}{}\doxysection{Constraints}\label{doc_decisions_commit_function_md_autotoc_md1708}

\begin{DoxyItemize}
\item It should be possible for all existing plugins to run normally
\end{DoxyItemize}\hypertarget{doc_decisions_commit_function_md_autotoc_md1709}{}\doxysection{Assumptions}\label{doc_decisions_commit_function_md_autotoc_md1709}
\hypertarget{doc_decisions_commit_function_md_autotoc_md1710}{}\doxysection{Considered Alternatives}\label{doc_decisions_commit_function_md_autotoc_md1710}
\hypertarget{doc_decisions_commit_function_md_autotoc_md1711}{}\doxysection{Decision}\label{doc_decisions_commit_function_md_autotoc_md1711}
Committing will no longer be done by {\ttfamily \mbox{\hyperlink{group__kdb_ga11436b058408f83d303ca5e996832bcf}{kdb\+Set()}}}. Instead, the functionality will be implemented by its own function, {\ttfamily kdb\+Commit()}.\hypertarget{doc_decisions_commit_function_md_autotoc_md1712}{}\doxysection{Rationale}\label{doc_decisions_commit_function_md_autotoc_md1712}
The implementation of this function is a prerequisite for a future improvement to backends, which will enable greater amounts of plugins in a single backend and plugin statelessness.

Currently, plugins combining the {\ttfamily commit} role with a different role (such as resolver plugins which combine the {\ttfamily setresolver} and {\ttfamily commit} roles) need to distinguish between these roles and track their state.\hypertarget{doc_decisions_commit_function_md_autotoc_md1713}{}\doxysection{Implications}\label{doc_decisions_commit_function_md_autotoc_md1713}
