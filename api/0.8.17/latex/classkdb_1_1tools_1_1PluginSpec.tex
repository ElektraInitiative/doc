Specifies a plugin by its name and configuration. 

\begin{DoxyInvariant}{Invariant}
name is valid (nonempty, starts with a-\/z, then a-\/z\+\_\+0-\/9) 

refname is valid (same as above or a size\+\_\+t number) 
\end{DoxyInvariant}
