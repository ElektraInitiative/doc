{\ttfamily kdb mountpoint-\/info \mbox{[}mount point\mbox{]}}\hypertarget{doc_help_kdb-mountpoint-info_md_autotoc_md2625}{}\doxysection{DESCRIPTION}\label{doc_help_kdb-mountpoint-info_md_autotoc_md2625}
This command will print information about the version, default resolver and default storage plugin. When a mount point is given as argument additional information about the mount point (e.\+g. configuration) is printed.\hypertarget{doc_help_kdb-mountpoint-info_md_autotoc_md2626}{}\doxysection{EXAMPLES}\label{doc_help_kdb-mountpoint-info_md_autotoc_md2626}
