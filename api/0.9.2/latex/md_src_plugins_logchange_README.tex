The purpose of this plugin is to demonstrate how one can be notified of every removed, added or changed key easily.\hypertarget{md_src_plugins_logchange_README_autotoc_md391}{}\doxysection{Usage}\label{md_src_plugins_logchange_README_autotoc_md391}
Prints every added, changed or deleted key on the console. To use it, add it during mounting\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{kdb mount logchange.dump user/logchange dump logchange}
\end{DoxyCode}


Configure the plugin with {\ttfamily log/get=1} to enable printing when configuration is loaded. For example, {\ttfamily kdb gmount logchange log/get=1}.
