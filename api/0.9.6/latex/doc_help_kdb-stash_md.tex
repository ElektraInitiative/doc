{\ttfamily kdb stash}\hypertarget{doc_help_kdb-stash_md_autotoc_md2555}{}\doxysection{DESCRIPTION}\label{doc_help_kdb-stash_md_autotoc_md2555}
This command will stash away the {\ttfamily system}, {\ttfamily user} and {\ttfamily spec} configuration, i.\+e. it releases all mount points and reset Elektra to a clean state. Afterwards a timestamp which can be used to restore everything will be printed.

The backup will be done in the {\ttfamily /var/tmp} directory, so make sure the backup is not deleted.\hypertarget{doc_help_kdb-stash_md_autotoc_md2556}{}\doxysection{EXAMPLES}\label{doc_help_kdb-stash_md_autotoc_md2556}
